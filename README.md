# Career Guidance System

An interactive web-based Career Guidance System that helps students choose suitable career paths based on their interests, skills, and preferences.

## Features
- User registration & login
- Career interest selection
- Course recommendations
- University suggestions
- Certifications list
- Helpful external resources
- Interactive chatbot guidance
